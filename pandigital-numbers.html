In Mathematics a number is said to be Pandigital if it contains all the digits from 1 to N atleast once.Where N is the number of digits in the number. For example: N=5 then 12345 is a Pandigital number but 11234 is not as it does not contains 5

#code to check whether a number is Pandigital or not 
# The technique imployed is a brute force one

n=int(raw_input("enter the number of digits"))
number_check=raw_input("enter the number")
digits=[str(i) for i in range(1,n+1)]
flg=1
for i in digits:
    if i in number_check:
        pass
    else:
        flg=0
        break
if flg==0:
    print "number is not pandigital"
else:
    print "number is pandigital"
